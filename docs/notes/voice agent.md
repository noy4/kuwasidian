## LiveKit 採用事例まとめ

- **OpenAI (ChatGPT)**: [OpenAI and LiveKit partner to turn Advanced Voice into an API](https://livekit.com/blog/openai-livekit-partnership-advanced-voice-realtime-api) - ChatGPTの「高度な音声モード」を支える技術として採用され、Realtime APIを共同提供。
- **xAI (Grok)**: [Introducing the Grok Voice Agent API in partnership with xAI](https://livekit.com/blog/xai-livekit-partnership-grok-voice-agent-api) - Grokのリアルタイム音声対話機能を実現するインフラとして採用。
- **Open Interpreter (01)**: [Livekit Server - 01 Documentation](https://01.openinterpreter.com/server/livekit) - 音声操作デバイス「01」のリモートコマンド実行用通信サーバーとして採用。
- **Skydio**: [Use Case: Robotics - Skydio flies with LiveKit](https://livekit.com/use-cases/robotics) - ドローンの遠隔監視やテレオペレーションにおける低遅延映像伝送に採用。
- **Polymath Robotics**: [How Polymath Robotics Uses LiveKit to Remotely Operate Heavy Machinery](https://livekit.com/blog/how-polymath-robotics-uses-livekit-to-remotely-operate-heavy-machinery) - 自律走行重機の遠隔操作用ビデオストリーミング基盤として採用。
- **Salesforce**: [Welcome, LiveKit!](https://salesforceventures.com/perspectives/welcome-livekit/) - リアルタイムAIワークロードの基盤としてSalesforce Venturesからの出資・採用。
- **Playback**: [How Playback built a global stadium for watching sports together](https://livekit.com/blog/how-playback-built-a-global-stadium-for-watching-sports-together) - スポーツ同時視聴プラットフォームの大規模WebRTCインフラとして採用。
- **Assort Health**: [Healthcare Case Study](https://livekit.com/customers) - 医療ワークフローにおける音声AIエージェントの構築基盤として採用。
