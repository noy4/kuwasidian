- 飯を食う（必須課題）

- 楽しむ

自分の進みたい方向が明確なら、進むのに必要なツールを探し、選び、習得して進む
- どう食っていきたいか
- どうありたいか

見えていない場合、とりあえず基礎ツールを習得する。
その過程で方向性が見えればラッキー。
道が違うと感じたら都度修正
